⚡ TradeFaster
Мод для Minecraft [Fabric], который добавляет кнопку «Buy All» в торговлю с жителями. Больше не нужно кликать 16 раз подряд — одно нажатие скупает весь дневной лимит.

Как это работает
Открываешь торговлю с жителем, выбираешь нужный трейд, нажимаешь ↻ Buy All — мод сам быстро скупает всё, что есть в ассортименте на сегодня. Кнопка неактивна (серая), пока ты не выберешь трейд, чтобы случайно не нажать.

Установка
Установи Fabric Loader 0.19.3+ — скачать
Установи Fabric API для Minecraft 26.1.2 — скачать
Скачай TradeFaster.jar и кинь в .minecraft/mods/
Всё
Что важно знать
Только клиент — мод не нужно ставить на сервер. Работает на любом сервере, где ты играешь.
Безопасно — мод просто быстро нажимает за тебя, никаких эксплойтов. Сервер получает обычные пакеты торговли.
Умная остановка — мод сам прекратит покупку, когда товар закончится или не хватит изумрудов.
Не теряет предметы — каждый клик дожидается ответа сервера перед следующим.
